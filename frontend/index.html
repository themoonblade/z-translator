Z-TRANSLATOR

Comment utiliser Z-TRANSLATOR

Z-TRANSLATOR offre 3 fonctionnalités principales :

  • Traduction : Traduire du texte entre 6 langues (Auto, FR, EN, ES, DE, IT, PT)
  • Résumé : Créer un résumé concis du texte
  • Réécriture : Améliorer la clarté et le professionnalisme du texte

Utilisation : Saisissez votre texte, choisissez la fonction, et cliquez sur le bouton correspondant.

0 chars

Traitement de fichiers

Formats supportés : DOCX, PDF, TXT, HTML

Fonctions disponibles :

  • Translate File : Traduit et télécharge le fichier dans son format original
  • Summarize File : Crée un résumé du contenu
  • Rewrite File : Améliore la qualité rédactionnelle

Timeout : Ajustez le délai d'attente selon la taille du fichier (1 min à 2 heures)

Drag & Drop or Click to Upload

Supported formats: .docx, .pdf, .txt, .html

Documentation API

URL de base : http://localhost:8001

Documentation Swagger : http://localhost:8001/docs

Tags disponibles : Models, Translation, Summarization, Rewriting

Endpoints principaux

GET /models

Liste tous les modèles Ollama disponibles

POST /translate/text

Traduit du texte entre deux langues

Paramètres : text, source_lang, target_lang, model

POST /summarize/text

Résume du texte de manière concise

Paramètres : text, model

POST /rewrite/text

Réécrit du texte pour améliorer la clarté

Paramètres : text, model

Notes importantes

  • Tous les endpoints /file acceptent un paramètre timeout (60-7200s)
  • Les fichiers traduits sont retournés dans leur format original
  • Les résumés et réécritures de fichiers retournent du JSON
  • Documentation complète sur /docs
